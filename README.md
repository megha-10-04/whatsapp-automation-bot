# WhatsApp Automation Bot 🤖

This project sends WhatsApp messages automatically using Python.

## 🚀 Features
- Send messages using phone number
- Opens WhatsApp Web automatically
- Sends multiple messages without reopening

## 🛠️ Technologies Used
- Python
- PyWhatKit
- PyAutoGUI
- Pyperclip

## 📦 Installation
pip install pywhatkit pyautogui pyperclip

## ▶️ How to Run
python whatsapp_sender.py
